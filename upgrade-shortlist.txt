# 🚀 WhisperPost

A modern **anonymous messaging platform** where users can receive messages from anyone through a public link — built with a focus on **privacy, simplicity, and premium user experience**.

---

## 🌐 Overview

WhisperPost allows users to:
- Create an account
- Share a unique public link
- Receive anonymous messages
- Manage and interact with messages in a clean, modern interface

---

## ✨ Key Features

### 🔗 Public Anonymous Messaging
- Each user gets a **shareable public link**
- Anyone can send messages without revealing identity

---

### 🎨 Premium Dark UI (Midnight Glass Theme)
- Fully redesigned **dark theme UI**
- Glassmorphism-based message cards
- Gradient backgrounds (dark → neon accents)
- Smooth hover and glow effects

---

### ✨ Advanced Animations
- Staggered message entry animations
- Expand-on-click message cards
- Smooth swipe-to-delete interaction
- Unread message highlight with subtle pulse effect

---

### 💌 Mood-Based Messages (Unique Feature)
Senders can choose a “mood” while sending messages:
- 💖 Love
- 🔥 Confession
- 😂 Funny
- 😈 Secret
- 💭 Advice

Each mood dynamically changes the **visual style of the message card**, making the experience more expressive and engaging.

---

### 🖼️ Premium Message Card System
- Gradient-based message cards
- Highlight unread messages
- ❤️ Love / 🗑️ Delete actions
- 📌 Pin important messages
- 📷 Download messages as PNG

---

### 📊 Smart Analytics Dashboard
- Total messages received
- Daily and weekly stats
- Clean and minimal data visualization

---

### 🔔 Notification Indicator
- Real-time unread message counter
- Subtle animated notification badge

---

### 🎁 Hint-Based Anonymity
- Optional hint system for senders:
  - Friend
  - Same class
  - Stranger

Maintains anonymity while increasing curiosity and engagement.

---

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS, Framer Motion  
- **Backend:** Next.js API Routes  
- **Authentication:** JWT (Cookie-based)  
- **Validation:** Zod  
- **Database:** MongoDB  

---
