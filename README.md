# WhisperPost

WhisperPost is a modern anonymous messaging platform that allows users to receive honest and anonymous messages through a personalized public link.

Simply create an account, share your unique profile link, and start receiving anonymous messages from friends, followers, or anyone on the internet.

---

## ✨ Features

### 🔐 Authentication

* Credentials-based Authentication
* Google OAuth Login
* Email Verification System
* JWT Session Management
* Protected Routes & Middleware

### 💬 Anonymous Messaging

* Receive anonymous messages through a public profile link
* No sender identity is revealed
* Clean and distraction-free messaging experience

### 📊 Dashboard Analytics

* Total Messages Count
* Daily Message Statistics
* Weekly Message Statistics
* Unread Message Tracking

### ❤️ Message Management

* Mark messages as favorites
* Delete unwanted messages
* Highlight unread messages
* Organize your inbox efficiently

### 📸 Story Card System

* Generate custom story cards
* Download story cards as images
* Share directly on social media
* Instagram Story integration support

### 🎨 Modern UI/UX

* Premium Dark Theme
* Fully Responsive Design
* Smooth Animations
* Glassmorphism Inspired Components
* Mobile-Friendly Experience

## 🏃 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/whisper-post.git
```

Navigate into the project:

```bash
cd whisper-post
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

in your browser.
